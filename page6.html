Helpful Study Resources For absolute beginners to Machine Learning!

As I said in my 'About Me' Page, one of the most important reasons to create this website is to share my experience with you. So in this post, I want to introduce some valuable and useful resources and free courses for learning Machine Learning. You should note that the content of this post will be updated regularly so some materials will be edited or replaced with a new one. Another point you should consider is that the content is not completely neutral because it is based on my experiences. I was not familiar with computer science, and even my bachelor's degree is not related to computer science fields, but sincerely I was a little familiar with C++ and writing pseudocode for simple algorithms. Don't worry even if you are not familiar with writing simple pseudocode; I'm here to introduce some courses and resources from beginner to professional. The only prerequisites to learn are passion and hard working.

Choosing a Proper Programming Language

The programming language I want to introduce is Python. Python will cover all your needs in data science and complicated concepts in AI. There are a wide variety of scientific packages, enormous free Tutorials, books, and users society for that. There are a lot of Python codes in Github and Python has active users that quickly respond to your problems and questions in StackOverflow. Almost all of the popular frameworks and packages in deep learning and big data supported Python. Besides, Python is not only a Scientific programming language, with Python you can writing server-side code in web programming and develop applications for platforms such as Windows, Android, IOS, etc.

Guido van Rossum author of the Python programming language (Source of photo)

Learning Python

For learning python, there are many resources, but I think this vast information and tutorials about that can confuse any students and any beginners for choosing a suitable way. For learning Python, begin with SOLOLEARN Tutorials which you can use as an application in Android and IOS or as a web application in a browser and even you can write your Python code and run it in the Sololearn environment without any Python installation on your machine. However, Sololearn has a weakness that only you will become familiar with the syntax, and you will not be a problem solver. You should know how to approach a problem and how to solve it based on simple steps. So what's the solution? You should use another tutorial as a supplementary.

UDACITY tutorials are perfect in a nutshell, complicated concepts taught simply, and you learn courses from practical and theoretical aspects.

my advice  

First of all, begin with SOLOLEARN tutorials and don't worry about forgetting tips and concepts, you should repeatedly review the course and using google as a help. Now it turns to use the UDACITY tutorial, for better understanding. you should solve course practices, search your problems or ask in discussion forums and if you find a better solution for a particular practice, share it with others.

In this way, you can find high self-esteem in python programming.

Math

You need to know and perceive convex optimization. As a result, you should study system equations, limits, derivative concepts.

In probability courses, Bayes theorem is widely have been used.

In statistics, you should be familiar with mean, median, variance, covariance, correlation, normal distribution, skewness, and the tendency.

You should have a good understanding of matrices and linear algebra concepts because it has been widely used in the machine learning domain, particularly Eigenvalues and Eigenvectors, which are essential for matrix decomposition in dimensionality reduction.

For the probability course, I read 'A First Course in Probability' by Sheldon Ross, for statistics I read 'John E. Freund's Mathematical Statistics with Applications' book. For Mathematica, I read 'Calculus a complete course' by Robert A. Adams and many books for Linear Algebra.

Don't worry; you don't need to read a lot of books like me; there are many free online courses and books for learning math, but we should choose the best ones. By this awesome channel, 3Blue1Brown, on youtube, you can watch and learn the concept of mentioned titles in math deeply and simply besides that by these books[1, 2] you can get the materials and content for prerequisites math for machine learning.  

My advice

Use the order of books[1, 2] content titles as a road map and for grasping the concepts use 3Blue1Brown or Khan Academy website, be aware that you should take notes from the lessons, after learning the concepts turn back to the books and solve some exercises from that (If you need further understanding I advise searching on youtube instead of web searching, there are many useful channels and videos which you can find).

Numpy, Pandas, and Visualizing 

For mathematical operations like working with matrices, using logarithm and derivative you need to use Numpy(Numerical Python) package in Python, and for reading your dataset and manipulating it, creating Dataframes and edited it, you need Pandas package. Totally in the preparation phase of your dataset for applying a machine learning algorithm, you need Pandas and Numpy packages.

We usually use visualization techniques in two-stage, first before deploying any machine learning algorithms or any manipulation in our data to get a better insight into that and Second after applying machine learning algorithms or any manipulation to visualize our results graphically. To do this in Python usually, we use Matplotlib and Seaborn(Seaborn has been developed based on Matplotlib library).

To learning Numpy and Pandas, you can use chapters 1 through 3 Python Data Science Handbook by Jake VanderPlas, or you can use these links[Numpy, Pandas] to have a quick grasp. For learning visualization by Matplotlib and Seaborn, you can use chapter 4 of the mentioned book or use this link.

My advice

To learning Numpy and Pandas, first of all, study Chapters 1-3 of the book I mentioned, particularly in the first chapter, you learn how to work with the IPython kernel and Jupyter Notebook environment. Then you can use the mentioned links [Numpy, Pandas] as reviewing sources. Don't worry if you forget codes and techniques, you don't study books and tutorials for learning all the materials, but you study those for knowing what keywords you should search in google for your problems.

For learning Matplotlib and Seaborn basics use this link, and you can see Chapter 4 of the mentioned book as a valuable source of beautiful codes. In future tutorials in machine learning that I am going to introduce, you will learn more professional techniques and practical tips about Numpy, Pandas, Matplotlib and seaborn.

Machine Learning

In university, we studied 'Data Mining: Concepts and Techniques' book by Jiawei Han which covers many algorithms and techniques, but it based on my experience, it's not a deep book, and I don't like it. The concepts are confusable and sloppy but in contrast 'Pattern Recognition and Machine Learning' by professor Christopher M. Bishop is the most valuable machine learning book that you can find, but for studying that, you should have a good understanding of math. Besides that 'A Course in Machine Learning' by Hal Daumé III is sweet and lovely. machine-learning algorithms in this book explained simply and enjoyably, and I promise you are not going to be confused by studying this book. Thank you Hal Daume :-).

In tutorials, I pick up three of the best ones. These tutorials are the Microsoft Machine Learning course Python edition in EDX, Udacity Machine Learning course, and Andrew Ngs' Machine Learning course on Coursera. Among these three tutorials that I mentioned, Microsoft course in EDX is better than the others, because based on my work experience in the industry, it is more practical, and its road map is more rational than the others. You can learn how to approach a machine learning problem from beginning to end. Also, it has a lot of exercises and useful python codes that by digging and investigating that codes, your programming level going to grow significantly (especially your skills in working with Numpy, Pandas, Matplotlib and Seaborn Packages).

However, the Microsoft Machine Learning course has some disadvantages. The most important of those is, Some materials explanations are boring and Incomprehensible.

My advice

Start with Microsoft Machine Learning course in EDX, if you don't understand any of the materials and lessons get help from Andrew Ng and Udacity courses (you can find the course videos directly on YouTube[1,2] and course notes in Websites). For example, when I couldn't perceive the Logistic Regression concepts, I used the Andrew Ngs course explanation as complementary, and for the decision tree, I used the Udacity decision tree lesson. Even if you similar to me like studying books, you can get help from Hal Daume or prof Bishops' books based on your needs and ability.

So as you notice, every source has its advantages and disadvantages, and you should not limit yourself to a particular source.
